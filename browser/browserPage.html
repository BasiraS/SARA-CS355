Browser

The Browser Object Model (BOM) allows JavaScript to "talk to" the browser.

Navigator

The navigator object contains information about the visitor's browser.

Application Name

It uses the appCodeName property to return the application name of the browser.

Engine

It uses the product property to return the product name of the browser engine.

Version

It uses the appVersion property to return the version information about the browser.

Agent

It uses the userAgent property to return the user-agent header sent by the browser to the server.

Platform

It uses the platform property to return the browser platform (operating system).

Language

It uses the language property to return the browser's language.

Window

The window object represents the browser's window.

Inner Height

It uses the innerheight property to represent the height of browser's window (in pixels).

Inner Width

It uses the innerwidth property to represent the width of the brower's window (in pixels).

Take a Look
Screen

The screen object contains information about the user's screen.

Width

It uses the width property to return the width of the visitor's screen in pixels.

Height

It uses the height property to return the height of the visitor's screen in pixels.

Available Width

It uses the availWidth property to return the width of the visitor's screen, in pixels, minus interface features like the Windows task bar.

Available Height

It uses the availHeight property to return the height of the visitor's screen, in pixels, minus interface features like the Windows task bar.

Color Depth

It uses the colorDepth property to return the number of bits used to display one color.

Pixel Depth

It uses the pixelDepth property to return the pixel depth of the screen.

Take a Look
Location

The location object contains information about the current page address (URL).

Href

It uses the href property to return the href (URL) of the current page.

Hostname

It uses the hostname property to return the domain name of the web host.

Pathname

It uses the pathname property to return the path and filename of the current page.

Protocol

It uses the protocol property to return the web protocol used (http: or https:).

Geolocation

The geolocation object represents a user's position by utlizing the HTML Geolocation API.

Position

It uses the getCurrentPosition method to return the user's position.

Latitude

It uses the position.coords.latitude property to represent the latitude of the user's position.

Longitude

It uses the position.coords.longitude property to represent the longitude of the user's position.
